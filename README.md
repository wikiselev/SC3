### Installation

```{R}
install.packages("devtools")
devtools::install_github("wikiselev/cs3")
```

### Input format

The input expression matrix should be of the following format:

gene1 1 2 3 4 5  
gene2 1 2 3 4 5  
gene3 1 2 3 4 5  

It should not have a header and separators should be either spaces or tabs.
